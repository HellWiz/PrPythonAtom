по адресу /get_classifier_result/<version>
читает из полученного запроса json-контент
В случае, если version==1, то возвращается json с версией и полем predict из входящего jsonа {"version":1, "predict": <predict_value>}
В случае, если version==0, то возвращается json с версией и полем old_predict из входящего jsonа {"version":0, "predict": <old_predict_value>}
